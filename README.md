# 🛒 Amazon Clone

A simple Amazon-inspired e-commerce homepage built using **HTML**, **CSS**, and **JavaScript**. This project was created for learning front-end web development and demonstrates responsive layouts, basic styling, and JavaScript interactions.

---

## 📌 Features

- Responsive navigation bar
- Search bar UI
- Hero/Banner section
- Product cards
- Buy Now button
- Responsive design
- JavaScript alerts and search functionality
- Clean and beginner-friendly code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🚀 Getting Started

1. Clone this repository

```bash
git clone https://github.com/your-username/amazon-clone.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No additional installation is required.
